# 数学导航地图 · Math Atlas

> 按图索骥学数学：一张可交互的知识图谱，正向（从全景逐层探索每个数学分支）+ 反向（给定应用目标反查所需数学路径）。

专为**"想学数学但不知从哪学起、学了能干什么"**的人设计——尤其是理工背景但基础模糊、想要成人化、目标导向讲解的自学者。

## 这是什么

一个纯前端单文件交互图谱（`math-atlas.html`，浏览器直接打开即可用，无需安装、无需联网）：

- **86 个数学节点**，按 7 大学习层级组织（基础 → 分析 / 代数 / 几何 / 概率统计 / 离散 / 应用）。
- **点击任意节点**：看到一句话定义、为什么学它、前置依赖、学了能去哪、推荐教材、知识点清单。
- **聚焦模式**：选中一个节点后，前置节点用青色连线标出、后续节点用橙色连线标出，一眼看清"先学什么、后学什么"。
- **目标路径**：25 条预设路径（机器学习、量子计算、密码学、信号处理等），告诉你为达某目标要走过哪些节点。
- **进度追踪**：勾选已掌握的知识点/节点，数据存本地浏览器，可导入导出备份。

## 谁会需要它

- 理工科学生，想补某块数学但课本太厚、不知重点。
- 转行 / 自学者，有应用目标（"我想搞懂机器学习需要什么数学"）却不知路径。
- 想快速建立数学全局观、再决定深入方向的人。

## 怎么用

1. 下载 `math-atlas.html`，双击用浏览器打开（推荐 Chrome / Edge）。
2. 全景图浏览各分支，或点击右上角"目标路线"选一个应用方向。
3. 点击节点进入聚焦模式，按青→当前的顺序学。
4. 详情面板可勾选知识点标记掌握进度。

## 内容质量

每个节点都标注了质量等级（详情面板的徽章）：

| 等级 | 含义 |
|---|---|
| **A 权威** | 有官方课标 / MSC 背书（基础层 18 节点） |
| **B 已核对** | 经公认教材核对（大学层节点） |

详见 [`docs/content-quality-standard.md`](docs/content-quality-standard.md)。
